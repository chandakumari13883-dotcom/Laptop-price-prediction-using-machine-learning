
# Laptop Price Prediction

This project predicts laptop prices based on specifications using Machine Learning.

## Dataset
Features used:
- Company
- TypeName
- Inches
- ScreenResolution
- CPU
- RAM
- Memory
- GPU
- OS
- Weight

Target:
- Price

## Steps Performed
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Train-Test Split
5. Model Training
6. Model Evaluation
7. Price Prediction

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab
